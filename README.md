# Get latest minecraft bedrock server download link
 
# Features
 
You can easily run Minecraft Bedrock Server.
 
# Requirement
 
golang 1.19
 
# Installation
 
```bash
sudo pacman -Syyu --noconfirm
sudo pacman -S go --noconfirm
sudo go build ./run.go
sudo chmod +x ./run
```
 
# Usage
 
```bash
sudo ./run
```
