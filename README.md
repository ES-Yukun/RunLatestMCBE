# Get latest minecraft bedrock server download link
 
# Features
 
You can easily get the download link for Minecraft Bedrock Server.
 
# Requirement
 
golang 1.19
 
# Installation
 
```bash
pacman -Syyu --noconfirm
pacman -S go --noconfirm
go build ./get.go
chmod +x ./get
```
 
# Usage
 
```bash
./get
```
